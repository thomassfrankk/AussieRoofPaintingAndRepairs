# Aussie Roof Painting & Repairs

Landing page for Aussie Roof Painting & Repairs.

## Stack
- Static HTML/CSS/JS
- Hosted on Netlify
- Contact form via Netlify Forms

## Deployment
Linked to Netlify — pushes to `main` auto-deploy.

## Structure
```
├── index.html          Main landing page
├── success.html        Form submission thank-you page
├── netlify.toml        Netlify config & caching headers
├── css/style.css       Styles
├── js/main.js          Scroll animations
└── images/
    ├── logo.png        Company logo
    └── hero.jpg        Hero image
```
